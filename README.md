# Student Performance Dashboard

A working student performance dashboard built with plain HTML, CSS, and JavaScript.

## Features
- Track marks, attendance, syllabus progress, and exam dates
- Auto-calculate overall score, attendance, and completion
- Identify high-risk subjects
- Show priority study list and recommendations
- Save data in browser localStorage
- No build command needed

## Run
Open `index.html` in a browser.

## Deploy
Upload `index.html`, `styles.css`, and `script.js` to a GitHub repo, then import it into Vercel.
